# v1.0 NLP LM

## Objective

`v1.0` trains a character-level GPT-nano language model on Tiny Shakespeare.
Lineage: Scratch run.

## Config Contract

- Train config: `configs/nlp_lm/v1.0_train.yaml`
- Smoke config: `configs/nlp_lm/v1.0_smoke.yaml`
- Control config: `configs/nlp_lm/v1.0_control.yaml`

## Expected Artifacts

- Best checkpoint: `artifacts/runs/v1.0/checkpoints/v1.0_train_best.pt`
- Train summary: `artifacts/runs/v1.0/metrics/train_summary.json`
- Eval summary: `artifacts/runs/v1.0/metrics/eval_summary.json`

## Metrics

- Primary: **BPB** (bits per byte) — lower is better
- Secondary: cross-entropy loss, perplexity
